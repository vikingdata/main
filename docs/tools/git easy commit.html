Purpose

I wanted to make it easy to save changes to the remote repository on GITHUB without using git commands or changing braches. Thus, this will add all files and upload them. For delete, or rename, the changes will be reflected at the next time you run the command. All comitted changes locally will synced to the remote repository. Most of the time, its just add. Thus this makes it easy.

Steps

  • Make Copy of git remote repository site. It it exists, just update it.  
  • Make sure we are not in git or we are in the master branch.
  • Rsync just the changes that exist on your local repository that does NOT exist in the copy. 
  • "'git add" all missing files on copy and not not include empty directories. Also filter out files we do not want. 
  • "git commit"
  • push the changes to the remote repository.
