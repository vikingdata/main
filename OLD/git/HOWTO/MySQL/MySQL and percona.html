MyRocks Engine

  1. Links:
    1. Percona comes with MyRocks.
    2. http://myrocks.io/
    3. https://www.percona.com/blog/2016/10/24/announcing-myrocks-in-percona-server-for-mysql/
    4. https://en.wikipedia.org/wiki/Comparison_of_MySQL_database_engines
    5. https://www.percona.com/blog/2018/02/01/myrocks-engine-things-know-start/
    6. https://github.com/facebook/mysql-5.6/wiki
    7. https://www.percona.com/doc/percona-server/8.0/myrocks/limitations.html
    8. https://minervadb.com/index.php/2018/06/06/a-friendly-comparison-of-innodb-and-myrocks-performance/
    9. https://db-engines.com/en/system/MongoDB%3BRavenDB%3BRocksDB
  2. Purpose: High performance, used by facebook, replacement for innodb.
  3. It is a key-value store like mongo.
  4. Adv:
    1. Faster, compressed data, etc.
    2. Acid -- this isn't talk about much. It seems to be, but I really question it.
    3. transnational
      1. Limited to available memory.
      2. read-committed and repeatable-read
    4. Works with XA
  5. Dis:
    1. Be very very careful how you setup, tables, run multi-row transactions, etc. It is easy to get non-optimized performance
    2. No foreign keys
    3. No full text or spatial keys
    4. 100% rollback??????
      1. No one talks about this. Disturbing,
      2. TODO: test this out, test begin commit and rollback.
    5. 100% SQL??????
      1. Seems to be SQL compliant, but nobody talks about it.
      2. TODO: perform complicated queries.
    6. Online DDL not supported yet.
  6. Install: It comes with Percona by default.

Percona tools

Percona has tons of tools that are useful. Some more than others. Years ago I wrote tools when some of these tools did not exist. I like most of them but

  • Checksum is nice ---- but isn't as flexible as I would like it.
  • Analyzing slow logs could be pushed a lot farther and doesn't answer questions which solve all the problems I have.

But I am highly impressed with all the tools and I could not have done better with most of them,

  • https://www.percona.com/software/database-tools/percona-toolkit
    • Download, install, and study each tool.
