What window functions do.

Window functions allow you to show calculations after all the data has been read into the query. Normally queries read the data as is and cannot order and group the data until the end of the query. Data is scanned once and they group or ordered. You cannot normally show the results of a grouped function until all the rows have been grouped.

With window functions, it really just allows you to show the results of the grouped commands with each line of data. For example, I want to show the entire sum of money for each state, and everybody in the state. I want to show one person per line and the repeating sum of money the person is on that state. Thus, The group function is calculated and displayed for each person in the state. Normally is the sum the money for a state in it one one line. The state and then the sum. But with window functions you can apply the result of the sum to more than one line. For each person in the state. An example is shown below plus more window functions.
