Install

Java and python should be fine if installed on latest ubuntu. I went ahead and installed java with: apt install openjdk-8-jdk -y

Then I installed more stuff. 

  apt install apt-transport-https

  sh -c 'echo "deb http://www.apache.org/dist/cassandra/debian 40x main" > /etc/apt/sources.list.d/cassandra.list'
  apt update
  apt install Cassandra


    # Verify with commands
  nodetool status
  systemctl status cassandra

  # if needed
  service cassandra restart
