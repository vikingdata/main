Setup and Install

  •       curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo tee \
      /usr/share/keyrings/jenkins-keyring.asc > /dev/null

      echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
      https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
      /etc/apt/sources.list.d/jenkins.list > /dev/null

      apt-get update
      apt-get -y install jenkins
      apt-get -y  install openjdk-11-jdk
      
      # Note this password
      cat /var/lib/jenkins/secrets/initialAdminPassword
  • Go to your browser and use this password to start. My example : http://192.168.1.6:8080/ -- your computer will have a different ip address.
  • Select install suggested plugins. I will later install more like the ssh plugin.
  • Create a administrator account when the plugins are done.
  • Redo the install plugins if there were any errors. We can come back later and work on the plugins.
  • It will wask for the default url for jenkins. You should not need to change it. Save it.
  • Click on "Start using Jenkins".
