Adding swap file

Check if /swapfile exists in /etc/fstab. If it is not..... 
  swapoff -a
  dd if=/dev/zero of=/swapfile bs=1024 count=8048576
  mkswap /swapfile
  chmod 600 /swapfile
  echo "/swapfile    none    swap    sw    0    0" >> /etc/fstab
  swapon -a
  swapon -s
